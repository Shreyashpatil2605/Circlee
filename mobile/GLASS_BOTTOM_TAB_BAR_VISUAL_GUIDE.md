### Tab Press Flow

```
Time: 0ms
в”ңв”Җ User touches tab
в””в”Җ activeOpacity: 1 вҶ’ 0.7 (instant visual feedback)

Time: 50-100ms
в”ңв”Җ State updates
в”ңв”Җ activeIndex changes
в””в”Җ Component re-renders

Time: 100-200ms
в”ңв”Җ New active tab renders
в”Ӯ  в”ңв”Җ Icon color: rgba(255,255,255,0.6) вҶ’ #FF3B30
в”Ӯ  в”ңв”Җ Background appears: transparent вҶ’ rgba(255,255,255,0.2)

в”Ӯ  в””в”Җ Border visible: subtle вҶ’ clear
в”Ӯ

   в”ңв”Җ Icon color: #FF3B30 вҶ’ rgba(255,255,255,0.6)
   в”ңв”Җ Background fades: rgba(255,255,255,0.2) вҶ’ transparent
```
